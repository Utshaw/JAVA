# Math functions
```
System.out.println(Math.abs(-100)); // 100
System.out.println(Math.min(2, 3)); // 2
System.out.println(Math.max(2, 3)); // 3
System.out.println(Math.round(17.9)); // 18
System.out.println(Math.round(17.4)); // 17
System.out.println(Math.ceil(17.10)); // 18
System.out.println(Math.floor(17.9)); // 17
System.out.println(Math.random()); // [0.0, 1.0)
System.out.println(Math.pow(2, 3)); // 8.0
System.out.println(Math.sqrt(64)); // 8.0
```
